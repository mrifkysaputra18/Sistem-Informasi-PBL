📋 LAWS OF UX YANG DITERAPKAN:

1. ✨ VISUAL HIERARCHY
   - Statistics cards dengan number yang lebih besar (4xl font)
   - Color-coded icons dengan gradient backgrounds
   - Clear section headers dengan gradient banners
   - Primary actions lebih prominent (green button)

2. 🎯 FITTS'S LAW
   - Tombol "Upload Progress" lebih besar & mudah diklik
   - Action buttons dengan padding optimal (px-5 py-3)
   - Touch targets minimal 44px untuk mobile
   - Spacing yang cukup antar elemen

3. 🧩 HICK'S LAW
   - Simplified choices: Upload atau Detail saja
   - Progressive disclosure untuk group details
   - Tidak overwhelm user dengan terlalu banyak info

4. 📦 LAW OF COMMON REGION
   - Cards dengan borders & shadows yang jelas
   - Grouping information dalam sections terpisah
   - Color-coded status badges
   - Consistent spacing & alignment

5. 🎪 VON RESTORFF EFFECT
   - Tombol Upload Progress warna hijau (stand out)
   - Animated badges untuk status urgent
   - Ring effect untuk overdue targets
   - Crown icon untuk ketua kelompok

6. 💎 AESTHETIC-USABILITY EFFECT
   - Gradient backgrounds yang modern
   - Rounded corners (rounded-xl, rounded-2xl)
   - Smooth shadows & hover effects
   - Professional color palette

7. 📱 JAKOB'S LAW
   - Familiar card layout (seperti social media)
   - Standard icon usage (Font Awesome)
   - Conventional button placement
   - Expected interaction patterns

8. 🔄 PROGRESSIVE DISCLOSURE
   - Group details hidden by default
   - "Lihat Detail" button untuk expand
   - Show essential info first
   - Reduce cognitive load

9. 📢 FEEDBACK & VISIBILITY
   - Hover effects pada semua interactive elements
   - Loading animations (spin icon)
   - Status progress bars
   - Success/error message banners
   - Pulse animation untuk urgent items

10. 📐 LAW OF PROXIMITY
    - Related info grouped together
    - Consistent spacing hierarchy
    - Clear visual relationships
    - Logical information flow

⚡ ANIMATIONS & INTERACTIONS:

1. CARD ENTRANCE ANIMATION
   - Stats cards fade in dengan stagger effect
   - Each card delays 100ms from previous
   - TranslateY(20px) -> TranslateY(0)

2. HOVER EFFECTS
   - Cards: shadow-md -> shadow-xl
   - Icons: scale(1) -> scale(1.1)
   - Buttons: -translate-y-0 -> -translate-y-0.5
   - Border colors change on hover

3. PROGRESSIVE DISCLOSURE
   - Smooth slideDown animation
   - Icon rotation (0deg -> 180deg)
   - Text change: "Lihat Detail" <-> "Sembunyikan"

4. STATUS INDICATORS
   - Pulse animation untuk pending items
   - Spin animation untuk loading states
   - Bounce animation untuk alert badges

5. SMOOTH SCROLLING
   - scroll-behavior: smooth
   - Transition durations: 200-500ms
   - Easing functions: ease-out, ease-in-out

═══════════════════════════════════════════════════════════════════

📱 RESPONSIVE DESIGN:

BREAKPOINTS:
- Mobile: < 640px (sm)
- Tablet: 640px - 1024px (md, lg)
- Desktop: > 1024px (xl)

ADAPTATIONS:
✅ Stats cards: 1 col (mobile) -> 2 cols (tablet) -> 4 cols (desktop)
✅ Group info: Stack vertical (mobile) -> Grid 4 cols (desktop)
✅ Target cards: Full width (mobile) -> Side-by-side actions (desktop)
✅ Header: Stack vertical (mobile) -> Flex row (desktop)
✅ Buttons: Full width (mobile) -> Auto width (desktop)
✅ Text sizes: Smaller on mobile, larger on desktop
